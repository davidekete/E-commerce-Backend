# E-Commerce-Backend
> An e-commerce web application backend built with NodeJS.
> 

A simple e-commerce backend with more features being added over time.

## Technologies used

- NodeJS
- ExpressJS
- MongoDB
- Mongoose

## Features

- [x]  Fetch all products
- [x]  Get single product
- [x]  Create Order
- [x]  User Login/Signup
- [x]  User authorization
- [ ]  Add to cart
- [ ]  Checkout
